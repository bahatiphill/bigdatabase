# bigdatabase

Attempt to get useful data from google maps because it is fun

## Setup

```cli
pip3 install pipenv
```

```cli
git clone https://github.com/knowbee/bigdatabase
```

```cli
cd bigdatabase
```

```cli
pipenv shell
```

```cli
pipenv install
```
